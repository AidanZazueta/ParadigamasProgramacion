## Introducción
Para esta practica conoceremos los conceptos basicos para el trabajo con repositorios de git, tenemos como objetivo gestionar proyectos de una manera ordenada.
Ademas exploraremos el uso de GitHub que nos permite guardar nuestros repositorios en la nube, dentro de GitHub trabajaremos con Hugo, una herramienta que nos sirve para generar paginas estaticas en base a Markdown. 
## Desarrollo
Para comenzar se hizo la instalacion de git en nuestro sistema, ademas de la implentacion de la direccion de hugo como variable global. Despues de la implementacion de git en el sistema hacemos un `git init` dentro de nuestro portfalio, para comenzar. <br>
Posteriormente enlazamos git y github por medio de llaves ssh, ahora con el comando `git push` podemos subir lo que tenemos en nuestro repositorio de git a nuestro repositorio en la nube de github.
Una vez realizado lo anterior procedemos con la implementacion de Hugo en nuestro sistema, esto para poder tener una pagina estatica que muestre nuestros reportes de practica realizados con Markdown, para ello tenemos que implemtarlo localmente, para posterior subirlo a github y realice la accion.
